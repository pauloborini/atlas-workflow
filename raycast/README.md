# Raycast — Talos snippets

## Importar

1. Abra o Raycast (`⌘ Space`)
2. Busque **Import Snippets**
3. Selecione `talos-snippets.json` nesta pasta
4. Revise a lista e confirme

## Keywords (prefixo `!aw-`)

| Keyword | Uso |
|---------|-----|
| `!aw-claude` | Instalar marketplace + plugin no Claude (GitHub) |
| `!aw-claude-l` | Idem, pasta local no volume |
| `!aw-claude-up` | Atualizar marketplace Claude |
| `!aw-codex` | Marketplace Codex (GitHub) |
| `!aw-codex-l` | Marketplace Codex (local) |
| `!aw-codex-up` | `codex plugin marketplace upgrade` |
| `!aw-codex-rm-personal` | Remover marketplace `personal` antigo |
| `!aw-cd` | `cd` no repo |
| `!aw-push` / `!aw-pull` | Git sync |
| `!aw-wf-claude` | `/workflow claude full sprint …` |
| `!aw-wf-codex` | `workflow codex full …` |
| `!aw-install-all` | Claude + Codex de uma vez |

Placeholders Raycast: `{cursor}` posiciona o cursor; `{clipboard}` cola o clipboard.
